Deli Gateau

Web institucional para una empresa de repostería

Para este proyecto, tomé un diseño en Behance y haciendole algunas variaciones implementé los elementos necesarios de HTML, CSS , y se añadí interacciones y funcionalidades mediante JavaScript y Bootstrap. En la etapa de diseño y planificación, tomé algunos elementos del diseño original reelaborandolos, así como también utilicé JS para hacer más dinámica la navegación del sitio, considerando factores como la usabilidad y la accesibilidad.

Una vez planificada la estructura, comencé a implementarlo usando HTML, Bootstrap combinando con mi propio CSS. Buscando siempre que el código fuera limpio y semántico para que la página sea fácil de entender y modificar en el futuro. Luego añadí interacciones y animaciones con JavaScript y CSS. Por ejemplo, haciendo aparecer y desaparecer elementos de imagen o implementar un botón para subir al top de la página. Algunas de estas animacionesya estaban integradas con Bootstrap, como la galería de imágenes.

En resumen, este proyecto permite poner en práctica la planificación, diseño y desarrollo web, y crear una página funcional y atractiva que se puede utilizar para diversos propósitos.
